# electron-final-exercises

Este repositorio contiene una serie de ejercicios prácticos para aprender y practicar el desarrollo de aplicaciones con tecnologías como Ionic, Express.js y MongoDB (incluyendo MongoDB Atlas). Cada carpeta corresponde a un módulo o conjunto de ejercicios enfocados en diferentes aspectos del desarrollo full stack con electron.

## Estructura principal

- **01-Cart-ionic/**: Ejercicios y solución para crear un carrito de compras usando Ionic.
- **02-Cart-express-mongo/**: Ejercicios y solución para implementar un carrito de compras con Express.js y MongoDB.
- **03-Books-crud-atlas/**: Ejercicios y solución para CRUD de libros usando Ionic y MongoDB Atlas.

Cada carpeta incluye proyectos, instrucciones y recursos necesarios para completar los ejercicios.

---

**Nota:** Contenidos adaptados para el módulo de DI del CFGS DAM. Desarrollado solamente con fines didácticos.
