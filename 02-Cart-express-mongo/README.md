# 02-Cart-express-mongo

Esta carpeta contiene ejercicios y soluciones para implementar un carrito de compras utilizando Express.js y MongoDB.

## Estructura

- **express-mongo/**: Proyecto backend con Express.js y MongoDB. Incluye el archivo principal `server.js`, archivos públicos y la configuración necesaria en `package.json`.
- **ionic-cart-express/**: Proyecto frontend basado en Ionic que se conecta al backend de Express para gestionar el carrito de compras. Incluye archivos fuente, recursos y configuración.

Utiliza esta carpeta para practicar la integración de un carrito de compras con un backend en Node.js y una base de datos MongoDB. Ideal para subir al repositorio y documentar el avance del proyecto.
