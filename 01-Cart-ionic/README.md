# 01-Cart-ionic

Esta carpeta contiene ejercicios y soluciones relacionados con la creación de un carrito de compras utilizando Ionic.

## Estructura

- **cart/**: Contiene los enunciados y pasos de los ejercicios, organizados en archivos `README-CART-01.md` a `README-CART-06.md`.
- **ionic-cart-initial/**: Proyecto base de Ionic para comenzar los ejercicios. Incluye archivos fuente, datos de ejemplo y recursos de imágenes.
- **ionic-cart-solution/**: Solución completa del carrito de compras en Ionic, con el código final y recursos necesarios.

Utiliza esta carpeta para practicar, consultar los enunciados o revisar la solución propuesta antes de subir el repositorio a GitHub.
